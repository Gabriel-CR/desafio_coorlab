## Implementação

### Tecnologias utilizadas

- Vue.js
- Styled Components

### Como executar

- Clone o repositório
- Instale as dependências com

```bash
cd code
npm install
```

- Execute a api com

```bash
npm run api_serve
```

- Execute o projeto com

```bash
npm run serve
```
